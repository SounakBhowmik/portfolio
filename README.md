# Quantum Scientist Portfolio

A Next.js (App Router) + TypeScript + Tailwind portfolio/blog with a research-forward, quantum-themed design language.

## Design system

- **Heading font:** Crimson Pro (serif, scholarly hierarchy)
- **Body font:** Inter (clean sans for readability)
- **Code font:** JetBrains Mono
- **Accent color:** Quantum teal (`--accent` in `src/styles/globals.css`)
- **Token location:** Update color tokens in `src/styles/globals.css` and font setup in `src/app/layout.tsx`

### Dependency notes

- No new runtime dependencies were added; typography is handled through custom global prose styles.

## Development

```bash
npm install
npm run dev
```
